    Yes24 Movie web

    YES24 영화예매어플을 리디자인한 고정형 모바일웹.

    working time : 2020.05~06

    degree of participation : Design(100%), Publishing(70%)

    Tool : 제이쿼리, html, css, 피그마, 포토샵, 일러스트

    Library Used : slick

    Available browsers : chrom, safari

    영화예매만을 원하는 고객을 타겟으로, 사용되는 목적에 충실할 수 있도록 심플하게 디자인 하였습니다. 메인화면에 slick을 사용한 슬라이드를 포인트로 고객이 최신 영화를 바로 접근 할수 있도록 하였고, 슬라이드에 버튼을 통해, 빠른 예매가 가능하도록 디자인하였습니다. 불필요한 메뉴를 숨기고, 필요한 메뉴는 탭메뉴를 활용하여, 고객의 편의성을 높였습니다.
